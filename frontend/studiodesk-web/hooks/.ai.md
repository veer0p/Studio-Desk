## Key Hooks

### `useAdminAuth()`
**File**: `use-admin-auth.ts`
**Purpose**: SWR-backed admin auth hook for platform admin panel

**Parameters**: None

**Returns**:
```typescript
{
  admin: AdminRecord | null;
  isLoading: boolean;
  isError: boolean;
  isAuthenticated: boolean;
  isSuperAdmin: boolean;  // derived: admin.role === 'super_admin'
  logout: () => Promise<void>;
  mutate: () => void;
}
```

**Calls To** (Level 1):
- `getAdminMe()` → `../lib/admin-api/.ai.md` → `GET /api/v1/admin/auth/me`
- `adminLogout()` → `../lib/admin-api/.ai.md` → `POST /api/v1/admin/auth/logout`
- `swr` → `useSWR('/api/v1/admin/auth/me', fetcher)` with 60s deduping

**Called By** (Level 1):
- Admin layout/sidebar → `../app/(admin)/.ai.md`
- Admin dashboard → `../app/(admin)/.ai.md`
- Studio detail page → `../app/(admin)/.ai.md`

### `useAuth()`
**File**: `use-auth.ts`
**Purpose**: SWR-backed auth state hook providing user, studio, and member context across the app

**Parameters**: None

**Returns**:
```typescript
{
  user: User | null;
  studio: { id: string; name: string; slug: string; plan_tier: string; onboarding_completed: boolean; brand_color?: string; logo_url?: string } | null;
  member: { id: string; role: string; display_name: string; profile_photo_url?: string } | null;
  isLoading: boolean;
  isError: boolean;
  isOwner: boolean;        // derived: member.role === 'owner'
  brandColor: string;      // studio.brand_color || '#1A3C5E'
  mutate: () => void;      // SWR manual revalidation
}
```

**Calls To** (Level 1):
- `getMe()` → `../lib/.ai.md` → `GET /api/v1/auth/me`
- `swr` → `useSWR('/api/v1/auth/me', fetcher)` with caching and 401 handling

**Called By** (Level 1):
- Dashboard layout → `../app/.ai.md`
- `GreetingSection` → `../components/.ai.md`
- Any component needing auth state across all domains → `../components/.ai.md`
