## Purpose
Public gallery API client — typed fetchers for gallery metadata and photos.

## Key Functions

| Function | Parameters | Returns | Endpoint |
|---|---|---|---|
| `fetchPublicGallery(slug)` | `{ slug: string }` | `GalleryData` (metadata + first 48 photos) | GET `/api/v1/gallery/[slug]` |
| `fetchGalleryPhotos(slug, page?)` | `{ slug, page }` | `{ photos, total, page, totalPages }` | GET `/api/v1/gallery/[slug]/photos` |
| `verifyClientPin(slug, pin)` | `{ slug, pin }` | void | POST `/api/v1/gallery/[slug]/verify` |

### Types

| Type | Fields |
|---|---|
| `GalleryPhoto` | `id`, `immich_asset_id`, `filename`, `mime_type`, `taken_at?`, `width`, `height`, `is_video`, `thumb_url`, `download_url` |
| `GalleryMetadata` | `name`, `status`, `total_photos`, `face_clusters[]`, `studio{ name, logo_url, brand_color }`, `event{ event_type, event_date }` |
| `GalleryData` | `metadata`, `photos[]` |
