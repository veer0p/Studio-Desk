## Key Functions

### API Client (`api.ts`)
**Purpose**: Central data-fetching layer — 30+ fetchers, 20+ mutations, normalization helpers

**Fetchers** (read-only, GET):
| Function | Returns | Endpoint |
|---|---|---|
| `fetchBookingsList(url)` | `BookingListResult` | `GET /api/v1/bookings` |
| `fetchBookingDetail(url)` | `BookingSummary` | `GET /api/v1/bookings/[id]` |
| `fetchClientsList(url)` | `ClientListResult` | `GET /api/v1/clients` |
| `fetchClientDetail(url)` | `ClientDetail` | `GET /api/v1/clients/[id]` |
| `fetchDashboardOverview(url)` | `DashboardOverview` | `GET /api/v1/dashboard/overview` |
| `fetchDashboardToday(url)` | `DashboardToday` | `GET /api/v1/dashboard/today` |
| `fetchFinanceSummary(url)` | `FinanceSummary` | `GET /api/v1/finance/summary` |
| `fetchInvoicesList(url)` | `InvoiceListResult` | `GET /api/v1/invoices` |
| `fetchInvoiceDetail(id)` | Invoice + lineItems + payments | `GET /api/v1/invoices/[id]` |
| `fetchGalleriesList(url)` | `GallerySummary[]` | `GET /api/v1/galleries` |
| `fetchGalleryDetail(url)` | `GalleryDetail` | `GET /api/v1/galleries/[id]` |
| `fetchContractsList(url)` | `ContractListResult` | `GET /api/v1/contracts` |
| `fetchProposalsList(url)` | `ProposalListResult` | `GET /api/v1/proposals` |

| `fetchAnalyticsRevenue(period)` | Revenue chart data | `GET /api/v1/analytics/revenue` |
| `fetchBillingInfo(url)` | `BillingInfo` | `GET /api/v1/settings/billing` |

**Mutations** (write operations, POST/PATCH/DELETE):
| Function | Purpose | Endpoint |
|---|---|---|
| `createBooking(data)` | Create client + booking | POST `/api/v1/clients`, `/api/v1/bookings` |
| `updateBookingStage(id, stage)` | Change pipeline stage | PATCH `/api/v1/bookings/[id]/status` |
| `createClient(data)` / `updateClient(id, data)` | Client CRUD | POST/PATCH `/api/v1/clients` |
| `createInvoice(data)` / `updateInvoice(id, data)` | Invoice CRUD | POST/PATCH `/api/v1/invoices` |
| `sendInvoice(id)` / `deleteInvoice(id)` | Send/delete | POST/DELETE `/api/v1/invoices/[id]` |
| `recordPayment(data)` | Record payment | POST `/api/v1/payments` |
| `createGallery(data)` / `updateGallerySettings(id, data)` | Gallery | POST/PATCH `/api/v1/galleries` |
| `createContract(data)` / `createProposal(data)` | Create doc | POST `/api/v1/contracts`, `/api/v1/proposals` |
| `createAddon(data)` / `updateAddon(id)` / `deleteAddon(id)` | Addon CRUD | `/api/v1/addons` |
| `createPackage(data)` / `updatePackage(id)` / `deletePackage(id)` | Package CRUD | `/api/v1/packages` |
| `createTeamMember(data)` | Invite member | POST `/api/v1/team/invite` |
| `submitStudioProfile(data)` | Onboarding step 1 | POST `/api/v1/studio/onboarding/1` |
| `submitOwnerProfile(data)` | Onboarding step 2 | PATCH `/api/v1/auth/me` |

### Auth (`auth.ts`)
| Function | Parameters | Returns | Endpoint |
|---|---|---|---|
| `signIn(email, password)` | Credentials | `{ user, studio, member }` | POST `/api/v1/auth/login` |
| `signUp(email, password, fullName, studioName, studioSlug)` | Signup data | `{ user, studio }` | POST `/api/v1/auth/signup` |
| `signOut()` | None | Redirects | POST `/api/v1/auth/logout` |
| `getMe()` | None | `{ user, studio, member }` | GET `/api/v1/auth/me` |
| `forgotPassword(email)` | Email | void | POST `/api/v1/auth/forgot-password` |
| `resetPassword(password)` | New password | void | POST `/api/v1/auth/update-password` |

**Called By** (Level 1):
- Auth pages → `../app/.ai.md`
- `useAuth` hook → `../hooks/.ai.md`
