## Data Fetching Patterns
- **Server Components (pages)**: Fetch data, pass as props to shells
- **Client Components**: `useSWR(fetcher)` with deduping
- **Mutations**: Direct `fetch` calls from `../lib/.ai.md`, followed by `mutate()` for cache invalidation
- **List stability**: Detail panels fetch own data; list remains stable on URL param changes
