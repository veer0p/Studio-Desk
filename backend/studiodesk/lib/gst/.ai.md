## Purpose
Indian GST (Goods and Services Tax) calculations — CGST+SGST for intra-state, IGST for inter-state, at 18% default rate.

## Key Functions

### `detectGstType(studioStateCode, clientStateCode)`
**File**: `calculator.ts`
**Purpose**: Determines GST type based on state codes (same state = CGST+SGST, different = IGST)
**Parameters**: `{ studioStateCode: string, clientStateCode: string | null }`
**Returns**: `GstType` (`'cgst_sgst' | 'igst' | 'exempt'`)
**Async**: No

**Called By** (Level 1):
- `BookingService.createBooking()` → `../services/.ai.md`
- `calculateInvoiceTotals()` → `./calculator.ts`

### `calculateGst(subtotal, gstType, rate?)`
**File**: `calculator.ts`
**Purpose**: Calculates GST amount — split into CGST+SGST (half each) or single IGST
**Parameters**: `{ subtotal: number, gstType: GstType, rate?: number }` (default rate: 18)
**Returns**: `{ cgst: number, sgst: number, igst: number, total: number }`
**Async**: No

**Called By** (Level 1):
- `BookingService` → `../services/.ai.md`
- `PackageService` → `../services/.ai.md`
- `ProposalService` → `../services/.ai.md`
- `calculateInvoiceTotals()` → `./calculator.ts`

### `calculateInvoiceTotals(lineItems, gstType)`
**File**: `calculator.ts`
**Purpose**: Calculates invoice subtotal from line items + GST breakdown + grand total
**Parameters**: `{ lineItems: Array<{ quantity: number, unit_price: number }>, gstType: GstType }`
**Returns**: `{ subtotal, cgst, sgst, igst, gstTotal, grandTotal }`
**Async**: No

**Called By** (Level 1):
- `InvoiceService` → `../services/.ai.md`
- `invoiceRepo` → `../repositories/.ai.md`

### `getCurrentFinancialYear()` / `getFinancialYearRange()`
**File**: `calculator.ts`
**Purpose**: Returns current Indian financial year string (`FY2526`) or date range (Apr 1 – Mar 31)
**Parameters**: None
**Returns**: `string` | `{ start: Date, end: Date }`
**Async**: No

**Called By** (Level 1):
- `invoiceRepo` → `../repositories/.ai.md`

interface LineItem {
