## Purpose
WhatsApp messaging client supporting both Meta Graph API and third-party providers (Interakt, etc.). Auto-detects provider based on `WHATSAPP_API_BASE_URL`.

## Key Functions

### `WhatsAppClient` (class)
**File**: `client.ts`
**Purpose**: Sends WhatsApp messages via API with provider auto-detection

**Methods**:
| Method | Parameters | Returns | Async |
|---|---|---|---|
| `send(payload)` | `{ to, type: 'text'|'template', text?, templateName?, templateComponents? }` | `{ messageId: string }` | Yes |
| `sendTextMessage(to, text)` | `{ to: string, text: string }` | `{ messageId: string }` | Yes |
| `sendTemplateMessage(to, templateName, components)` | `{ to, templateName, components[] }` | `{ messageId: string }` | Yes |

**File**: `client.ts`
**Purpose**: Singleton instance of `WhatsAppClient`

**Data Schema**:

**Common Patterns**:
- Auto-detects Meta vs non-Meta: if URL contains `graph.facebook.com`, uses Meta Graph API format
- Otherwise falls back to generic vendor format (Interakt-compatible)
- Auth: Meta uses `Bearer` token, others use `x-api-key` header
